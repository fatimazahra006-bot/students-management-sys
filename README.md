# students-management-sys
Student Management System is a Python project that manages student records with a simple menu interface. It lets users add students with unique IDs, validate grades, calculate averages, and assign performance classes. Data is saved to text files, displayed on demand, and summarized in reports with highest, lowest, and class averages.



Student Management System :

---Overview

The Student Management System is a simple Python project designed to manage student records through a menu-driven interface. It demonstrates practical use of input validation, file handling, and modular functions.

---Features

* Add students with unique IDs, names, and grades.
* Validate grades to ensure they are between 0 and 20.
* Automatically calculate averages and assign performance classes (A–F).
* Save student data into a text file for persistence.
* Display stored students by reading from the file.
* Generate reports with highest, lowest, and class averages.

---Technologies

* Python 3
* File handling (open, read, write)
* Lists and dictionaries
* Input validation

---File Structure

file name.txt → Stores student records.

report.txt → Stores statistical reports.

main.py → Main program with menu and functions.

---How to Run

* Clone or download the repository.

Run the program:

python main.py

"Use the menu to add students, display records, or generate reports".
